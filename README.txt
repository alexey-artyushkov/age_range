Программа для определения возрастных ограничений

Для запуска данный программы необходимо, чтобы у вас был установлен интерпритатор GNU-Prolog или SWI-Prolog.

ВАЖНО!!!
Файлы age_range.pro, ban.pro, six.pro, sixt.pro, twot.pro должны находиться в одной папке.
Итерпритатор запускайте только из той папки, в которой находится файл "age_range.pro".

Первый запуск
При первом запуске введите в окне интерпритатора "consult(age_range)." без кавычек. После того как вы ответите на вопросы, программа выдаст результат. Затем введите "save(save.db)." без кавычек. "save.db" - имя базы знаний, которая будет сохранена в той же папке что и файл "age_range.pro".

Последующие запуски
Для последующих запусков введите команду "consult(save.db).". "save.db" - имя базы знаний, которая будет загружена из той же папке что и файл "age_range.pro". Затем последовательность действия такая же, как и при первом запуске.

Ответы на вопросы
"yes." - да
"no." - нет
Все ответы вводятся без кавычек.
